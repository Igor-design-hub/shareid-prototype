ShareID Workflow & Pricing Prototype

Interactive prototype — open the HTML files below in any modern browser (Chrome, Safari, Firefox).

Start here

1. Workflow Builder → Build an onboarding or authentication flow by adding modules (Doc IDV, Face IDV, Auth). 2. Pricing Calculator → See price based on your flow, expected volume, and token pack.

How to use

  1. Open documentation.html — pick a preset or build a custom flow.
  2. Toggle between Onboarding and Authentication modes at the top.
  3. Click See pricing to jump to the calculator with your flow pre-loaded.
  4. In pricing.html, adjust the volume slider to estimate annual cost.
  5. Switch modes in pricing to compare onboarding vs. authentication costs.

Notes

  • Flows are saved in browser localStorage — they persist between sessions.
  • Both HTML files are fully self-contained; no server or internet required after download.
  • Keep the illustrations/ folder next to the HTML files.
